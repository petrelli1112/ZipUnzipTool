Zip / Unzip tool
================

This is a zip/unzip tool supported by [Karchive](https://projects.kde.org/projects/frameworks/karchive).

## 支持中文的压缩包的解压缩和包含中文的文件名的压缩包
Usage
=====

```cpp

ZipUnzip tool;
tool.unzip("zddhub.zip", "zddhub/");
tool.zip("zddhub/", "zddhub.zip");
```
